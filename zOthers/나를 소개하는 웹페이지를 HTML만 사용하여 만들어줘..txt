나를 소개하는 웹페이지를 HTML만 사용하여 만들어줘.

조건

- CSS 3 사용
- HTML5 문법을 사용한다.
- <!DOCTYPE html>부터 전체 코드를 작성한다.

페이지에는 다음 내용을 포함한다.

1. 제목(h1)
   - "○○○의 자기소개"

2. 자기소개 문단(p)

3. 프로필 이미지(img)
   - 이미지가 없으므로 placeholder 이미지를 사용한다.

4. 나의 취미
   - ul, li 태그를 사용한다.

5. 내가 좋아하는 음식
   - ol, li 태그를 사용한다.

6. 내가 자주 방문하는 사이트
   - a 태그를 사용하고 새 창에서 열리도록 한다.

7. 간단한 자기소개 표
   - 이름
   - 나이
   - 직업
   - 거주지역
   table 태그를 사용한다.

8. 마지막에 "연락하기" 버튼을 하나 추가한다.

사용한 주요 태그에 한 줄씩 주석으로 설명도 달아줘.
