샘플 사이트: https://personal-introductio-0nu7.bolt.host/
----------------------------------------------------
프롬프트
당신은 10년 이상의 경력을 가진 프론트엔드 웹 개발자이자 UI/UX 디자이너입니다.

HTML, CSS, JavaScript만 사용하여 감성적인 카페 홍보 홈페이지를 제작해주세요.

==============================
프로젝트명
==============================

별빛카페(Star Coffee)

==============================
목표
==============================

카페를 처음 방문하는 고객이
카페의 분위기와 메뉴를 살펴보고,
위치를 확인하고,
문의까지 할 수 있는 랜딩페이지를 제작합니다.

==============================
디자인 컨셉
==============================

- 따뜻하고 감성적인 분위기
- 브라운 + 베이지 계열 색상
- 여백이 많은 깔끔한 디자인
- 고급 카페 느낌
- 반응형 웹
- 부드러운 스크롤 효과
- 카드형 UI
- 둥근 버튼
- 그림자 효과
- Google Font 사용
- Font Awesome 아이콘 사용

==============================
메뉴 구성
==============================

1. 상단 메뉴(Header)

- 로고
- HOME
- MENU
- ABOUT
- GALLERY
- LOCATION
- CUSTOMER

스크롤하면 상단 메뉴가 고정되도록 구현합니다.

==============================
1. 랜딩 페이지(Hero)

화면을 가득 채우는 메인 이미지

카페 이름

한 줄 소개

대표 버튼

- 메뉴 보기
- 오시는 길

버튼 클릭 시 해당 영역으로 이동합니다.

==============================
2. 메뉴 소개(Menu)

카드 형태로 표시

다음 메뉴를 포함합니다.

- 아메리카노
- 카페라떼
- 바닐라라떼
- 카푸치노
- 티라미수
- 치즈케이크

각 카드에는

- 이미지
- 메뉴명
- 설명
- 가격

이 포함됩니다.

마우스를 올리면 카드가 살짝 확대되는 효과를 추가합니다.

==============================
3. 카페 소개(About)

카페 소개 글

대표 이미지

카페 특징

- 직접 로스팅
- 최고급 원두
- 무료 와이파이
- 단체석
- 주차 가능

아이콘과 함께 보기 좋게 표현합니다.

==============================
4. 갤러리(Gallery)

카페 사진 6장을 Grid 형태로 배치합니다.

사진을 클릭하면 크게 볼 수 있도록 Lightbox 효과를 구현합니다.

이미지는 무료 Placeholder 이미지를 사용합니다.

==============================
5. 오시는 길(Location)

Google Map을 iframe으로 삽입합니다.

아래 정보를 함께 표시합니다.

주소

전화번호

영업시간

주차안내

==============================
6. 고객센터(Customer)

게시판 형태의 테이블을 생성합니다.

컬럼

번호

제목

작성자

작성일

조회수

샘플 게시글 5개를 작성합니다.

아래에는

문의하기 버튼을 추가합니다.

==============================
7. Footer

- 카페명
- 주소
- 전화번호
- 이메일
- SNS 아이콘
- Copyright

==============================
기술 요구사항
==============================

HTML5

CSS3

JavaScript(ES6)

한 개의 HTML 파일에서 실행 가능하도록 작성합니다.

CSS는 style 태그에 작성합니다.

JavaScript는 script 태그에 작성합니다.

==============================
추가 기능
==============================

- 부드러운 스크롤
- 메뉴 클릭 시 해당 영역 이동
- Scroll Animation
- Hover Animation
- 모바일 대응
- 코드에 충분한 주석 작성

==============================
출력 형식
==============================

전체 HTML 코드를 처음부터 끝까지 작성합니다.

중간 생략 없이 완전한 코드만 출력합니다.
