# HTML/CSS 학습 예제 모음

이 저장소는 HTML 기본 태그와 간단한 CSS 스타일링을 연습하기 위한 예제 파일들로 구성되어 있습니다.

## GitHub 저장소

- https://github.com/swanbaek/basicWeb_osan.git

## 파일 구성

- `page01.html` : 제목, 문단, 줄바꿈, 주석 등 HTML 기초
- `page02_text.html` : 텍스트 관련 태그(h1~h6, i, em, strong, address, blockquote 등)
- `page03_img.html` : 이미지 삽입(`img`), `alt`, 크기 조절, `figure/figcaption`
- `page04_link.html` : 링크(`a`), 내부/외부 이동, 이미지맵(`map`, `area`)
- `page05_list.html` : 목록 태그(`ol`, `ul`, `dl`)와 중첩 리스트
- `page06_iframe.html` : `iframe` 사용 예제(내부 문서/유튜브 임베드)

### sample 폴더

- `sample/sample01.html` : 카페 가입 폼(기본 HTML 버전)
- `sample/sample02.html` : 카페 가입 폼(스타일 적용 버전)
- `sample/sample02.css` : sample02.html에 연결된 스타일 파일
- `sample/sample02_desc.css` : sample02.css의 학습용 주석 확장 버전

## 실행 방법

1. 원하는 `.html` 파일을 브라우저에서 열어 확인합니다.
2. VS Code Live Server 확장을 사용하면 변경 사항을 더 편하게 확인할 수 있습니다.
3. `sample/sample02.html`은 같은 폴더의 `sample02.css`를 참조하므로, 두 파일의 상대 경로를 유지하세요.

## 학습 추천 순서

`page01.html` → `page02_text.html` → `page03_img.html` → `page04_link.html` → `page05_list.html` → `page06_iframe.html` → `sample/sample01.html` → `sample/sample02.html`
